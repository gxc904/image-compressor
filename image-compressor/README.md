# Image Compressor（纯前端版）

这是一个纯前端图片压缩网站：上传图片后，浏览器在本地用 Canvas 重新编码来生成压缩预览，并支持下载压缩后的图片。

## 如何运行

1. 直接双击打开：`index.html`
2. 如果你的浏览器对 `file://` 有限制，也可以在该目录启动一个最简单的本地静态服务器后再打开：
   - Python：`python -m http.server 5500`
   - 然后访问：`http://localhost:5500/index.html`

## 使用说明

- 拖拽或点击上传图片
- 调整“输出格式 / 最大边 / 质量”
- 页面会自动更新压缩预览
- 点击“下载压缩后的图片”即可保存
